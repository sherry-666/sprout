# Sprout

Sprout is an AI-native day care management system designed to streamline communication between school administrators, teachers, and parents.

## Development

> **⚠️ IMPORTANT CODE UPDATE POLICY**
>
> Any update to the code needs to be accompanied by a review of the PRD (`doc/prd.md`) and TDD (`doc/tdd.md`).
> If your code changes introduce new functionality, alter existing behavior, or change the architecture, you **MUST** update the PRD and TDD accordingly to keep them in sync with the codebase.
